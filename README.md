# 💈 Greaser - Landing Page

Landing page moderna y responsive para una barbería, desarrollada con HTML, CSS y buenas prácticas de frontend.

## 🚀 Demo

👉 https://superlative-zuccutto-88cb42.netlify.app/

---

## 🧠 Objetivo del proyecto

Construir una landing page profesional aplicando:

* HTML semántico
* CSS organizado
* Responsive design (mobile first)
* Buenas prácticas de UI/UX
* Estructura limpia y escalable

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3 (Flexbox + Grid)
* Google Fonts

---

## ✨ Características

* Navbar sticky
* Hero section con layout en 2 columnas (desktop)
* Sección de servicios con cards
* Call to Action claro
* Formulario de contacto (UI)
* Diseño responsive
* Animaciones suaves (hover & transitions)

---

## 📱 Responsive Design

El sitio está optimizado para:

* Mobile
* Tablet
* Desktop

Aplicando enfoque **mobile-first**.
